🎵 48그룹 곡 분석

AKB48 · SKE48 · NMB48 · HKT48 · STU48

679곡

아키모토 야스시 프로듀스 전곡 데이터 기반

1. 분석 개요

본 분석은 아키모토 야스시가 프로듀스한 48그룹(AKB48, SKE48, NMB48, HKT48, STU48)의 전체 679곡을 대상으로 합니다. 각 그룹의 음악적 특성, 가사 테마, 시대별 변화를 AI 기반으로 분석했습니다.

📌 주요 발견

  • 48그룹은 그룹별로 명확히 구분되는 음악 스타일 보유
  • AKB48은 '연애송'과 '졸업송' 두 가지 스타일로 뚜렷이 구분됨
  • NMB48은 가장 독립적인 클러스터 형성 (쿨한 연애관)
  • HKT48은 수줍은 고백 스타일의 독자적 영역
  • 시대가 지남에 따라 '사랑' → '성장', '미래' 키워드로 변화

2. 분석 방법론

2.1 데이터 수집

다음 소스에서 Selenium 기반 웹 크롤링을 통해 데이터를 수집했습니다:

  • 나무위키 디스코그래피 - 각 그룹의 싱글 목록, 발매일, 센터, 선발멤버 정보
  • 48pedia.org - 나무위키에서 누락된 메타데이터 보완 (일본어→한글 변환 포함)
  • AKBLyrics 프로젝트 - AKB48 전곡 가사 원문 및 번역
  • Uta-Net, J-Lyric - 자매그룹(SKE48, NMB48, HKT48, STU48) 가사

2.2 AI 기반 가사 분석

Google Gemini API (gemini-2.0-flash 모델)를 사용하여 각 곡의 가사를 분석했습니다:

  • 테마 추출 - 곡의 핵심 주제를 한 문장으로 요약
  • 분위기 분류 - 밝음, 슬픔, 희망적, 청춘, 사랑 등 감정 태그
  • 키워드 추출 - 가사에서 핵심 키워드 5-7개 자동 추출
  • 타겟층 분석 - 10대, 졸업생, 연인 등 대상 청중 파악
  • 계절감/스타일 - 곡의 계절 배경 및 가사 스타일 특징

2.3 텍스트 임베딩 및 시각화

곡 간 유사도 분석을 위해 다음 기술을 적용했습니다:

  • Gemini text-embedding-004 - 각 곡의 분석 결과(테마, 키워드, 요약)를 768차원 벡터로 변환
  • t-SNE 차원 축소 - 고차원 임베딩을 2차원으로 축소하여 곡 분포 시각화
  • K-Means 클러스터링 - AKB48 곡을 '연애송'과 '졸업송' 두 그룹으로 자동 분류
  • 코사인 유사도 - 임베딩 벡터 간 유사도 계산으로 유사곡 탐색

3. 그룹별 키워드 분석

Gemini AI가 각 곡의 가사를 분석하여 추출한 키워드와 분위기입니다.

그룹곡 수주요 키워드주요 분위기

SKE48 94곡 (타이틀 35) 사랑, 짝사랑, 고백, 미래, 청춘 희망적, 청춘, 밝음
NMB48 106곡 (타이틀 32) 사랑, 미래, 꿈, 성장, 짝사랑 밝음, 희망적, 슬픔
HKT48 71곡 (타이틀 15) 사랑, 짝사랑, 미래, 청춘, 이별 밝음, 슬픔, 청춘
STU48 63곡 (타이틀 12) 사랑, 꿈, 미래, 성장, 응원 희망적, 밝음, 애절함

💡 그룹별 특징

  • AKB48: 가장 다양한 스펙트럼 - 연애, 졸업, 응원 전 영역
  • SKE48: 짝사랑, 청춘 중심의 직접적 감성
  • NMB48: 쿨한 연애관, 독립적 스타일
  • HKT48: 수줍은 고백, 귀여운 이미지
  • STU48: 바다/세토우치 감성, 진지한 메시지

4. 곡 분포 시각화 (t-SNE)

각 곡의 테마, 키워드, 분위기를 768차원 벡터로 임베딩한 후 t-SNE 알고리즘으로 2차원에 투영했습니다. 가까이 위치한 곡들은 유사한 테마와 감성을 가집니다. 마우스를 올리면 곡 정보를 확인할 수 있습니다.

차트 읽는 법

  • 점의 위치 - 가까운 점 = 유사한 곡. 멀리 떨어진 점 = 다른 스타일의 곡
  • ⭐ 별 마커 - 타이틀곡 (싱글 A면곡)
  • ● 원형 마커 - 커플링곡 및 앨범 수록곡
  • 색상 - 그룹별 상징색 (AKB48 핑크, SKE48 오렌지, NMB48 노랑, HKT48 검정, STU48 파랑)

분포 해석

AKB48 (연애송 클러스터) - 차트 우측에 분포. 「ヘビーローテーション」「フライングゲット」「恋するフォーチュンクッキー」등 밝고 직접적인 사랑 고백, 여름/청춘을 노래한 곡들이 모여있습니다.

AKB48 (졸업송 클러스터) - 차트 좌측에 분포. 「桜の花びらたち」「GIVE ME FIVE!」「365日の紙飛行機」등 졸업, 이별, 성장을 테마로 한 감성적인 곡들이 모여있습니다.

SKE48 - AKB48 연애송 클러스터와 인접. 짝사랑, 고백 테마가 많아 유사한 영역에 분포하지만 더 직접적이고 열정적인 스타일로 약간의 독자 영역 형성.

NMB48 - 독립적인 클러스터 형성. 쿨하고 도시적인 연애관, 「僕はいない」「欲望者」등 다른 48그룹과 차별화된 스타일.

HKT48 - 수줍은 고백, 귀여운 이미지의 곡들. 「好きになる時、その瞬間を」「桜、みんなで食べた」등 풋풋하고 수줍은 감성의 독자적 영역.

STU48 - 바다/세토우치 감성. 「風を待つ」「暗闇」등 진지하고 서정적인 메시지. 다른 그룹과 가장 뚜렷하게 분리된 독자적 클러스터.

🔍 흥미로운 발견

  • AKB48의 이중성: K-Means 클러스터링 결과, AKB48은 '연애송'과 '졸업송' 두 개의 뚜렷한 서브그룹으로 분리됨
  • NMB48의 독자성: 다른 48그룹과 가장 멀리 떨어진 독립적 클러스터 - 쿨한 연애관이 특징
  • HKT48 이상치: 「Dangerous Key」는 HKT48 곡이지만 STU48 클러스터에 위치 - 강렬하고 도전적인 스타일
  • 타이틀곡 집중도: 각 그룹의 타이틀곡(⭐)은 해당 그룹 클러스터의 중심부에 위치하는 경향

5. 시대적 변화 분석

2006년 AKB48 데뷔부터 현재까지 약 20년간 48그룹 음악이 어떻게 변화해왔는지 분석합니다.

5.1 초창기 (2006-2010)

주요 키워드: 사랑, 용기, 희망, 꿈, 고백

특징: AKB48 단독 활동기. 직접적이고 솔직한 사랑 고백, 밝은 희망의 메시지가 주를 이룸. 「会いたかった」「ヘビーローテーション」등 아이돌 연애송의 정석을 확립한 시기.

대표곡: 会いたかった, BINGO!, 大声ダイヤモンド, ヘビーローテーション

5.2 전성기 (2011-2015)

주요 키워드: 여름, 청춘, 짝사랑, 성장, 두근거림

특징: SKE48, NMB48, HKT48 등 자매그룹 본격 활동. 여름을 배경으로 한 청춘 감성이 폭발. 밝고 풋풋한 짝사랑, 학창시절의 추억을 노래한 곡이 대거 등장. 48그룹 전체 곡 수가 급증한 황금기.

대표곡: フライングゲット, Everyday、カチューシャ, 真夏のSounds good!, 恋するフォーチュンクッキー

5.3 다양화기 (2016-2020)

주요 키워드: 성장, 후회, 이별, 미래, 변화

특징: 단순한 연애송에서 벗어나 성숙한 테마 등장. '후회', '이별' 같은 어두운 감정도 다루기 시작. STU48 합류로 '바다', '세토우치' 같은 지역 감성 추가. 그룹별 개성이 더욱 뚜렷해진 시기.

대표곡: 365日の紙飛行機, LOVE TRIP, Teacher Teacher, 風を待つ (STU48)

5.4 현재 (2021-2025)

주요 키워드: 추억, 미래, 꿈, 새로운 시작, 응원

특징: 회고와 전망의 균형. 과거를 돌아보는 '추억' 테마와 미래를 향한 '희망' 메시지가 공존. 코로나 이후 '연결', '함께' 같은 키워드 증가. 각 그룹이 자신만의 확고한 정체성을 구축.

대표곡: 根も葉もRumor, 元カレです, ジワるDAYS

📈 전체 변화 요약

  • 테마 변화: 단순 연애 → 성장/졸업 → 회고/미래
  • 감정 변화: 밝음 일색 → 슬픔/후회 추가 → 다양한 감정 스펙트럼
  • AKB48 분화: 초기 통합 스타일 → '연애송'과 '졸업송' 두 갈래로 분화
  • 그룹 개성화: 유사한 스타일 → 각 그룹만의 독자적 영역 구축
